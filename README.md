# ToxicAI Code Reviewer 

An AI-powered code reviewer that delivers brutally honest, emotionally damaging, and completely useless feedback.

Built for the DEV April Fools Challenge.

---

## Features

- Paste your code → get roasted instantly
- Multiple personality modes:
  - 💀 Brutal
  - 🧠 Passive Aggressive
  - 🤖 Overly Supportive
- Fake analysis metrics:
  - Vibe Score
  - Ego Risk Level
  - Stack Overflow Dependency
- Dramatic loading states for maximum tension

---

## Tech Stack

- React (Vite + TypeScript)
- Tailwind CSS
- Google Gemini API

---

## Purpose

This project solves absolutely nothing.

It exists purely to:
- Demonstrate AI integration
- Emotionally damage developers

---

## Disclaimer

Do not use this tool if you are:
- Sensitive about your code
- Emotionally stable
- Expecting helpful feedback
